# Regression-for-SAT-and-Mosquitos
A review for regression line for SAT and GPA correlation as well as mosquito breeding sites and mosquito population

1. This project observes and analyzes the regression for two sets of data that being the relationship between SAT scores and GPA as well as the relationship between the number of mosquito breeding sites and the mosquito population.

2. The methods used for the for this analysis includes SAT-GPA regression line analysis same for mosquito breeding sites and population

3. There were a couple of key findings for these two models. Both included that there were significantly slight positive relationships between both data sets. However in one set of regression, the data breaks the assumption of linear regression.The SAT-GPA regression line breaks the assumption of linear relationships as the line is never linear at any points. There is extremely randomness between the data points. Even when looking at the residuals of each point, there is a drastic violation of the linear relationship assumption. The mosquito breeding relationship regression line does not violate any of the linear regression assumption and only does so when adding in an additional outlier that drastically changes the data set as seen in the image labeled mosquito development site with outlier.

4. The tools that were used in this assignment was R and data set provided in class namely the csv data set for SAT-GPA

5. Overall this assignment was extremely helpful in further understanding regression as well as analyzing graphs and how to properly use graphs for understanding regression relationships.
